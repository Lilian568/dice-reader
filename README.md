# Dice Reader

This repository contains code for some dice reading test software. Not recommended for serious use.   

See branch [Picamera-simple](https://github.com/nltabun/dice-reader/tree/Picamera-simple) for the simple Raspberry Pi Camera version   

See branch [pattern-cpp-v2](https://github.com/nltabun/dice-reader/tree/pattern-cpp-v2) for a more advanced version in C++ with pattern matching.
